# GitHub Upload Steps
1. Create public repo `enterprise-soc-l2-splunk-investigation-lab`.
2. Upload the CONTENTS of this folder, not the ZIP itself.
3. Add only screenshots you personally generated in Splunk.
4. Redact personal hostname/email/local paths/account/license details from screenshots.
5. Verify README, sample CSVs, and that no answer key or employer/customer claim is public.
6. Add the repository URL to the Projects section of your resume.
